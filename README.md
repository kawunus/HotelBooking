# HotelBooking

Веб-приложение для бронирования отелей. Проект предоставляет пользователям удобный интерфейс для поиска и бронирования номеров в отелях.

## Описание проекта

HotelBooking — это современное приложение для онлайн-бронирования гостиничных номеров. Проект разработан с использованием стандартных веб-технологий и предназначен для предоставления удобного пользовательского опыта при поиске и резервировании жилья.

## Используемые технологии

### Frontend

- **HTML5** — структурирование веб-страниц
- **CSS3** — стилизация и адаптивный дизайн
- **JavaScript** — интерактивность и логика приложения

### Зависимости

- **Lodash** (v4.17.23) — утилита для работы с массивами и объектами
- **Nodemon** (v3.1.11) — инструмент для автоматической перезагрузки при разработке

### Инструменты

- **Node.js** — среда выполнения JavaScript
- **npm** — менеджер пакетов

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/kawunus/HotelBooking.git
cd HotelBooking
```

2. Установите зависимости:

```bash
npm install
```

3. Для разработки с автоматической перезагрузкой используйте Nodemon:

```bash
npm run dev
```
